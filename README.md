## Run Locally

Clone the project

```bash
  git clone https://github.com/imohammedramadan/Sol-Chive.git
```

Go to the project directory

```bash
  cd sol-chive
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
